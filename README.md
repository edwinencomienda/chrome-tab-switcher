# Chrome Tab Window Switcher

This Chrome extension makes it easy to find and switch between your open tabs across all browser windows.

## What It Does

When you click the extension icon, a popup appears showing all your open tabs from every window. You can search through them and quickly switch to any tab you want.

## How to Use

1. Click the extension icon in your Chrome toolbar
2. Type in the search box to find specific tabs
3. Use arrow keys to navigate the list
4. Press Enter or click to switch to a tab

## Installation

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the extension folder
5. Pin the extension to your toolbar

## Why It's Useful

If you have many tabs open across different windows, this extension helps you find and switch to any tab instantly without having to hunt through multiple windows.
